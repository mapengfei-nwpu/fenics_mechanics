Scripts
=======

fenicshotools comes with two scripts that convert from a gmsh related file
to a FEniCS h5 file.

Geo to dolfin
-------------

.. argparse::
   :module: fenicshotools.scripts.geo2dolfin
   :func: get_args
   :prog: geo2dolfin

Gmsh to dolfin
--------------

.. argparse::
