Welcome to fenicshotools's documentation!
=========================================

fenicshotools is a small package for dealing with higher ordered
meshes in `FEniCS`_. fenicshotools wraps `Gmsh`_ and its mesh
generation functionality.

To download the source code or to report issues visit the `Bitbucket page
<https://bitbucket.org/peppu/fenicshotools>`_
